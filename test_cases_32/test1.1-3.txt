=== 3.2 控制流：if 多条件表达式 ===
// 测试if语句中包含逻辑与(&&)和逻辑或(||)的复合条件
// 输入两个整数m,k, 若 m<k 或 (m!=0 且 k==1) 则输出666
// 输入: 两个整数, 例如 0 1 (满足 k==1 且 m!=0...不满足,需 m<k)
main()
{
   int m,k,result;
   result = 666;
   m = read();
   k = read();
   if(m < k || m != 0 &&k == 1)
        write(result);
}
=== 输入: 1 4 ===
=== 预期输出: 666 ===
