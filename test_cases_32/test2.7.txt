=== 3.2 嵌套控制：if+for/while 选择计算奇偶和 ===
// 测试if嵌套for和while, 根据用户选择计算不同结果
// 输入N(范围1~100)和choice(1或2):
//   1: 求1~N奇数和
//   2: 求0~N-2偶数和
//   其他: 输出0
main()

{
  int i,N,sum = 0,choice=0;
  N = read();
  choice=read();
  if(choice == 1) {
      for(i=1;i<=N;i=i+1)
      {
         if(i%2 == 1)
	    sum = sum+i;
      }
  }
  else if(choice == 2){
      i=0;
      while(i<N){
      sum = sum + i;
      i = i + 2;
     }
   }
  write(sum);
}
=== 输入: 10, 1 ===
=== 预期输出: 25 ===
